web top

web top 은 비표준의 문제점 다양한 디바이스에 대응하는 서비스 제공을 위해 고객과 사용자를 중심으로 UI서비스를 통합하고, 웹 표준 애플리케이션을 개발 할 수 있는 단말 솔루션을 의미한다.

기존 단말의 기능과 전사적 뷰를 통합해 단일 서비스를 제공하고, 화면간 중첩기능을 제공으로 입력을 최소화한다.

web top의 기능

web top은 websquare 5 를 기반으로 작동된다.

.

기반 시스템의 성능과 안정성은 “web top에서 매우 중요한 요소이다. “web top”의 기반적 요소가 완전해도, 기반시스템이 불안정할 경우 HTML5 * 의 전환이 어렵다.web top 의 쓰임

web top은 간편시스템을 비롯한 디바이스 지원, 은행 디바이스 연계등 단말에 특화된 모든 기능을 제공한다.

웹 단말은 HTML5 기반의 웹스퀘어5로 구현돼 있어 하나의 공통 화면과 자바스크립트로 구현되는 비즈니스가 데스크톱 PC에서 태블릿, 스마트폰, 키오스크 등 다양한 디바이스에서 동일하게 실행된다.
